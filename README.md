# GO-Link-Shorte

Простой **link shortener**, написанный на **Go**.  
Мой **первый проект на Go**, делал для практики и понимания, как вообще работает Go + HTTP.

---

## Что это

GO-Link-Shorte умеет:
- сокращать длинные ссылки
- хранить их
- редиректить по короткому URL на оригинальный

Ничего лишнего - минимальный сервер + логика сокращения.

---

## Что использовал

- Go
- Go modules
- зависимость: `url-shortener`
